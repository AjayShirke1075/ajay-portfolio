## 🎨 Customization Tips

### Update Your Information
1. **Profile Image**: Replace `assets/images/profile/profile.jpg`
2. **Resume**: Update `resume/Ajay_Shirke_Resume.pdf`
3. **Email**: Change `ajayshirke@example.com` to your real email
4. **LinkedIn**: Add your LinkedIn profile URL
5. **Stats Numbers**: Update `data-target` values in hero stats
6. **Projects**: Add your actual project links

### Color Scheme
All colors use the cyan theme (#06b6d4). To change:
- Find and replace `#06b6d4` with your color
- Update gradient colors in CSS
- Adjust rgba values for consistency

### Add More Projects
Copy a `.project-card` div and modify:
- Change icon in `.project-overlay`
- Update title, description
- Modify tech stack tags
- Add your GitHub/demo links
