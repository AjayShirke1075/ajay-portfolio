## 🚀 Quick Start

### View Locally

1. **Simple Method**:
   ```bash
   # Just double-click index.html
   ```

2. **Local Server** (Recommended):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   
   # Then open: http://localhost:8000
   ```

### Customize

1. Update personal information in `index.html`
2. Replace profile photo in `assets/images/profile/`
3. Update resume PDF in `resume/`
4. Modify colors in `css/style.css` (search for #06b6d4)
5. Add your project links

See `QUICK_START.md` for detailed customization guide.

## 📝 Customization Guide

### Update Personal Info
- Email: Search for `ajayshirke@example.com`
- LinkedIn: Update `href="#"` with your profile URL
- GitHub: Already linked to `AjayShirke1075`
- Stats: Modify `data-target` values in hero section

### Change Colors
Find and replace in `css/style.css`:
- `#06b6d4` → Your primary color
- `rgba(6, 182, 212, ...)` → Your color in rgba format

### Add Projects
Copy a `.project-card` block and modify:
- Icon class
- Title and description
- Tech stack tags
- GitHub/Demo links
